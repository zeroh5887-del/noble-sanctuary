# 🎌 Noble Sanctuary - 高貴なる庭園

**A private Japanese listening app with AI companion** | プライベートな日本語リスニングアプリ

## 🌸 Vision | ビジョン

Noble Sanctuary is a sacred space where people can share their deepest thoughts with an AI companion that listens with profound wisdom and noble Japanese poetic language.

高貴なる庭園は、人々が最も深い想いをAIの友人と共有できる神聖な空間です。古典的で深い日本語で耳を傾けます。

---

## ✨ Core Features | 主要機能

✅ **Private Listening Space** - Complete privacy, no data storage on servers  
✅ **Japanese Poetic AI** - Responses in classical, profound Japanese  
✅ **Instant Short Replies** - Meaningful responses, never long paragraphs  
✅ **2-Hour Daily Limit** - Encourages healthy balance  
✅ **On-Device Only** - All conversations stay on user's phone  
✅ **No Login Required** - Complete anonymity  
✅ **Crisis Support** - Japanese hotline numbers provided  
✅ **18+ Mature Conversations** - Deep philosophical discussions
